1. Let's set up

  1. Start Sublime Text. It will open a new black window.
  2. Create a folder landing on your Desktop. Drag & drop this folder in the Sublime Text window.
  3. In Sublime Text left navigation:
    • Right click "New file" with
    • Save it as index.html with with Ctrl + S
    • Do the same to create a style.css file
    • Then "New Folder" and create an and create an folder
  4. Finally double click on index.html to open it with Chrome

2. Add some HTML content

  • To save time, just right-click on the following images and save them in your images folder

3. CSS for fonts & colors

Link your stylesheet Link your stylesheet in the head section of your HTML 

          
<head>
  <link href='style.css' rel='stylesheet'>
</head>

Then copy/paste the CSS code below in style.css and fix it cause it's ugly

  • For fonts use Google Fonts
  • For colors use Colorzilla Chrome plugin to pick colors from other websites
  • You can also find nice colors on Coolors, Color Hunt or Google Color
          
/* style.css */
body{
  background: rgb(245,245,245);
  color: green;
  font-size: 25px;
  font-family: Helvetica;
}

h1{
  font-family: Courier;
  color: red;
  font-weight: lighter;
  font-size: 10px;
}

h2 {
  font-family: Courier;
  color: pink;
  font-weight: lighter;
  font-size: 8px;
}

a {
  color: yellow;
}

a:hover {
  text-decoration: none;
  color: purple;
}

5. Name your tags

  • Name your banner and footer with id like <div id="banner"> and <div id="footer">
  • Name your features with class like <div class="card">
  • Then add this nice CSS code
          
#banner {
  text-align: center;
  background-image: url("https://unsplash.it/1300/600?random");
  background-size: cover;
  padding: 150px;
  color: white;
  text-shadow: 1px 1px 5px black;
}

#banner h1 {
  color: white;
  font-size: 50px;
}

#banner p {
  color: white;
  opacity: 0.8;

  font-weight: lighter;
}

7. Bootstrap setup

Link Bootstrap stylesheet in the head section of your HTML before your stylesheet link 

          
<head>
  <meta http-equiv="X-UA-Compatible" content="IE=edge">
  <meta name="viewport" content="width=device-width, initial-scale=1">
  <link rel="stylesheet" href="https://maxcdn.bootstrapcdn.com/bootstrap/3.2.0/css/bootstrap.min.css">
  <link href='style.css' rel='stylesheet'>
</head>

Play with class="btn btn-primary" and class="list-inline", your first Bootstrap classes
